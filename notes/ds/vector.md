# Vector notes

1) Vector access opertions : 

                push_back();
                pop_back();

2) Sorting :

                sort(vec.begin() + i , vec.begin() + j);
                where i is the start index and vec.begin() + j defines the before index of end index;


3) Reverse :

                reverse(vec.begin() + i, vec.begin() + j);
                same as the previous ones...;

4) Rotate :

                rotate();
